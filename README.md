## ✨ المميزات الرئيسية

### 👨‍🎓 للطلاب
- ✅ تسجيل دخول آمن برقم الهاتف
- 📺 مشاهدة الفيديوهات التعليمية
- 📖 تحميل المذكرات الدراسية (PDF)
- 🧪 حل الاختبارات والحصول على النتائج فوراً
- 📊 متابعة نسبة الإكمال والتقدم
- 💬 التواصل المباشر مع المعلم عبر WhatsApp

### 👨‍🏫 للمعلم (د. محمود)
- 🎬 نشر الفيديوهات والمذكرات الدراسية
- 📝 إنشاء اختبارات إلكترونية متكاملة
- 👥 إدارة وتفعيل حسابات الطلاب
- 📊 عرض إحصائيات شاملة عن أداء الطلاب
- 🎯 حجب الطلاب غير النشطين
- 🎨 تخصيص أسماء المواد والألوان
- 📈 رسوم بيانية تفاعلية لأداء الطلاب

## 🚀 التثبيت والتشغيل

### 1️⃣ استنساخ المستودع
```bash
git clone https://github.com/kimos65592/Doctor-Muhammad.git
cd Doctor-Muhammad
```

### 2️⃣ تثبيت المكتبات
```bash
npm install
```

### 3️⃣ تشغيل المشروع محلياً
```bash
npm start
```
سيفتح المتصفح على `http://localhost:3000`

### 4️⃣ بناء نسخة الإنتاج
```bash
npm run build
```

## 🗄️ بنية قاعدة البيانات Firebase

```
artifacts/
  ├── edu-platform-pro-v1/
  │   └── public/data/
  │       ├── users/           # بيانات الطلاب
  │       ├── contents/        # الفيديوهات والمذكرات

  │       ├── results/         # نتائج الاختبارات
```
